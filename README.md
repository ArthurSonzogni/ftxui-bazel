# FTXUI + Bazel example

Demonstrate how to build and run a simple terminal UI using [FTXUI](https://github.com/ArthurSonzogni/FTXUI) with [Bazel](https://bazel.build/).

**Run the example**

```bash
bazel run hello_ftxui_dom
```
